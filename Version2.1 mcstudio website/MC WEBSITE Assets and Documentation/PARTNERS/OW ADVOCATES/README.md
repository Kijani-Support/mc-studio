# OW ADVOCATES

This folder contains assets and documentation for the advocate program.

## Content

- Advocate program overview
- Eligibility and requirements
- Benefits and incentives
- Advocate resources and materials
- Success stories and testimonials
- Advocacy opportunities
- Recognition programs
- Contact information for advocates

## Subfolders

- **Images & Media/** - Advocate testimonials, program graphics, achievement badges
- **Text & Content/** - Program descriptions, advocate resources, guidelines

## Best Practices

- Highlight advocate success stories
- Provide clear program guidelines
- Include recognition opportunities
- Document benefits clearly
- Maintain regular communication
- Provide marketing materials
