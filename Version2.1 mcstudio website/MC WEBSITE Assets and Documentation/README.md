# MC Website Assets and Documentation

This directory contains all assets, media, and documentation for the MC Studio website v2.1.

## Directory Structure

### Main Sections

- **ABOUT US** - Company information, mission, team, and values
- **CONTACT** - Contact page assets and contact form documentation
- **FAQ** - Frequently asked questions and support information
- **HOME** - Homepage assets and main landing page content
- **IBM PRODUCTS** - IBM product solutions and offerings
- **LEGAL HEALTH CHECK** - Legal assessment and health check tools
- **PARTNERS** - Partnership programs and partner information
- **PROJECTS** - Project portfolio and project details
- **SERVICES** - Service offerings and service descriptions
- **STARTUPS** - Startup program information and directory
- **SUPPORT CENTER** - Customer support and help resources

## Content Organization

Each section typically contains:
- `Images & Media/` - Images, videos, icons, and multimedia assets
- `Text & Content/` - Text content, copy, and documentation

## Guidelines

- All media files should be organized by type (images, videos, etc.)
- Text content should follow consistent formatting standards
- Asset naming conventions should be descriptive and searchable
- Keep documentation updated when content changes
