# HOME - Images & Media

This folder contains all image and media assets for the Homepage.

## Content

- Homepage hero banner images
- Feature section images
- Service highlight graphics
- Partner logos
- Customer testimonial photos
- Company/brand imagery
- Icons and visual elements
- Background images
- Promotional graphics

## Organization

Files should be organized by:
- Section (hero, features, testimonials, etc.)
- Image type (banner, icon, thumbnail, etc.)
- Size/format variants (mobile, tablet, desktop)

## Technical Specifications

- Formats: WebP (preferred), PNG, JPG
- Hero banner: 1920x600px or responsive
- Thumbnail images: 400x300px minimum
- Icons: 64x64px to 256x256px
- Optimize for web: compress before uploading
