# ABOUT US - Images & Media

This folder contains all image and media assets for the About Us page.

## Content

- Company logo and branding assets
- Team member photographs
- Office and facility images
- Company culture and event photos
- Icons and graphics
- Promotional media

## Organization

Files should be organized by:
- Type (logos, photos, icons)
- Resolution and format
- Usage context (web, print, social media)

## Technical Specifications

- Web images: PNG, JPG, WebP formats
- Recommended resolution: 1920x1080px or higher
- File naming: descriptive-lowercase-with-hyphens.ext
