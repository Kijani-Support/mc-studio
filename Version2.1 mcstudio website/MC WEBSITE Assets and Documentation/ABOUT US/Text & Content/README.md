# ABOUT US - Text & Content

This folder contains all text content and documentation for the About Us page.

## Content

- Company description and elevator pitch
- Mission and vision statements
- Company values and culture information
- Team member bios and profiles
- Company history and timeline
- Press releases and news

## Format

Text content should be organized in:
- Markdown (.md) files for documentation
- Plain text (.txt) files for simple content
- JSON files for structured data if applicable

## Guidelines

- Write clearly and professionally
- Keep content up-to-date
- Use SEO-friendly keywords where appropriate
- Maintain consistent tone and voice
