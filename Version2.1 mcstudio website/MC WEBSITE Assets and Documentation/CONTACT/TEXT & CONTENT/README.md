# CONTACT - Text & Content

This folder contains all text content and documentation for the Contact page.

## Content

- Contact form field labels and placeholder text
- Contact information (addresses, phone numbers, emails)
- Department contact details
- Office hours and availability
- Contact page copy and introductory text
- Form validation messages
- Success and error messages

## Format

- Markdown files for longer form content
- Plain text files for short contact snippets
- JSON for structured contact data

## Important Notes

- Ensure all contact information is accurate and current
- Include response time expectations
- Provide clear instructions for users
- Make phone numbers clickable and formatted correctly
