# CONTACT - Images & Media

This folder contains all image and media assets for the Contact page.

## Content

- Contact page headers and banners
- Location maps and icons
- Social media icons
- Contact form graphics
- Department or team icons
- Placeholder images for contact methods

## Organization

Files should be named descriptively and organized by:
- Contact method type
- Image purpose (header, icon, background)
- Resolution

## Technical Specifications

- Web-optimized formats: PNG, JPG, SVG
- Icon size: 48x48px to 256x256px
- Banner size: 1920x600px or responsive dimensions
