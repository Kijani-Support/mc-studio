# STARTUPS

This section contains assets and documentation for the startup program.

## Subfolders

- **Individual Profiles/** - Individual startup profiles and company information
- **Join Ecosystem/** - Information on how to join the startup ecosystem
- **Portfolio Directory/** - Directory of startups in the portfolio
- **Success-Stories/** - Success stories and achievement highlights

## Purpose

This section showcases the startup program, portfolio companies, and startup success stories.

## Best Practices

- Keep startup information current
- Highlight success stories and metrics
- Document startup achievements
- Provide clear enrollment process
- Maintain startup profiles
- Include founder information
