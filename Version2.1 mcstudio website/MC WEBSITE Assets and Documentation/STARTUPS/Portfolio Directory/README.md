# Portfolio Directory

This folder contains directory information for all startups in the portfolio.

## Content

- Startup directory and listings
- Startup categorization and filtering
- Search functionality information
- Startup summary information
- Industry and sector classifications
- Startup status (active, acquired, etc.)
- Contact information listings
- Growth metrics and statistics

## Subfolders

- **Images & Media/** - Startup logos, category icons, directory graphics
- **Text & Content/** - Startup summaries, directory descriptions, classification information

## Best Practices

- Keep startup directory updated
- Enable searching and filtering
- Include high-quality logos
- Organize by industry/sector
- Provide summary information
- Include contact information
