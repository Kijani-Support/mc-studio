# SUPPORT CENTER - Images & Media

This folder contains all image and media assets for the Support Center.

## Content

- Support page headers and banners
- Help and support category icons
- Troubleshooting flowchart graphics
- Knowledge base graphics
- Support channel icons (email, phone, chat)
- FAQ section graphics
- Status page graphics
- Instructional and tutorial images

## Organization

- Organize by support category
- Name files based on usage
- Maintain consistent styling

## Technical Specifications

- Formats: PNG, JPG, SVG
- Icon size: 64x64px to 128x128px
- Banner size: 1920x600px or responsive
- Optimize for web loading speed
